##开发环境部署##

安装开发环境依赖

    npm install -g grunt-cli bower

安装nodejs依赖和bower依赖

    npm install && bower install

配置以下文件

    app/server/config.js

运行

    npm start

享受它吧！


##生产环境部署##

修改`NODE_ENV`为`production`

运行`grunt`构建js和css

启动项目
